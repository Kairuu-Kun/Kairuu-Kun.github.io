Fibonacci Sequence

Fibonacci refers to a sequence of numbers in which each number is the sum of the two preceding ones, usually starting with 0 and 1. The sequence goes 0, 1, 1, 2, 3, 5, 8, 13, 21, and so on. This sequence has numerous applications in various fields, including mathematics, art, and nature.

Lucas Sequence

The Lucas sequence is another sequence of numbers similar to the Fibonacci sequence. It starts with the seed values 2 and 1, and each subsequent term is the sum of the two preceding ones. The sequence goes 2, 1, 3, 4, 7, 11, 18, and so on.

Tribonacci Sequence

Similar to the Fibonacci sequence, the Tribonacci sequence is a series of numbers in which each number is the sum of the three preceding ones. It typically starts with the seed values 0, 0, 1. So, the sequence would go 0, 0, 1, 1, 2, 4, 7, 13, and so forth.

Collatz Sequence

The Collatz sequence, also known as the 3n + 1 sequence, is a mathematical sequence defined as follows: start with any positive integer n. Each subsequent term is obtained from the previous one using the following rules: if the previous term is even, the next term is one half of the previous term; if the previous term is odd, the next term is 3 times the previous term plus 1.
